## 📍 ২. লাইভ লোকেশন ট্র্যাকিং (কিভাবে কাজ করে)

### মূল কনসেপ্ট: "ক্রাউডসোর্সিং"

```
┌─────────────────────────────────────────────────────────────┐
│              লাইভ ট্র্যাকিং কিভাবে কাজ করে                    │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│   ধাপ ১: যাত্রীরা লোকেশন শেয়ার করে                           │
│   ┌─────────────────────────────────────────────────┐       │
│   │  👤 করিম (ট্রেনে আছে)  →  📍 GPS অন → শেয়ার      │       │
│   │  👤 ফাতেমা (ট্রেনে আছে) →  📍 GPS অন → শেয়ার      │       │
│   │  👤 রহিম (ট্রেনে আছে)  →  📍 GPS অন → শেয়ার      │       │
│   └─────────────────────────────────────────────────┘       │
│                         ↓                                    │
│   ধাপ ২: সার্ভার সব লোকেশন একত্রিত করে                       │
│   ┌─────────────────────────────────────────────────┐       │
│   │  📍 করিম: ২৩.৮৫, ৯০.৪০                          │       │
│   │  📍 ফাতেমা: ২৩.৮৬, ৯০.৪১                        │       │
│   │  📍 রহিম: ২৩.৮৪, ৯০.৩৯                          │       │
│   │          ↓                                       │       │
│   │  📊 গড় পজিশন: ২৩.৮৫, ৯০.৪০                      │       │
│   │  🎯 নির্ভুলতা: উচ্চ (৩ জন কন্ট্রিবিউটর)          │       │
│   └─────────────────────────────────────────────────┘       │
│                         ↓                                    │
│   ধাপ ৩: ম্যাপে দেখায়                                       │
│   ┌─────────────────────────────────────────────────┐       │
│   │  🗺️ ম্যাপে ট্রেনের পজিশন                        │       │
│   │     ● ← ট্রেন এখানে                              │       │
│   │     গতি: ৬৫ km/h                                │       │
│   │     নির্ভুলতা: উচ্চ ✅                           │       │
│   └─────────────────────────────────────────────────┘       │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

### স্পিড কিভাবে আসে?

```
যাত্রীর ফোনের GPS থেকে স্বয়ংক্রিয়ভাবে আসে:

📱 ফোনের GPS ডাটা:
├── 📍 Latitude (অক্ষাংশ)
├── 📍 Longitude (দ্রাঘিমাংশ)  
├── 🏃 Speed (গতি) ← এখান থেকে আসে
├── 🧭 Heading (দিক)
└── 📏 Accuracy (নির্ভুলতা)
```

### ✅ আপনার কিছু করতে হবে না!

লাইভ ট্র্যাকিং সিস্টেম সম্পূর্ণ তৈরি। যাত্রীরা যখন লোকেশন শেয়ার করবে, স্বয়ংক্রিয়ভাবে সব হবে।
